Java Backend API

Welcome to the official documentation for the Java Spring Boot REST API serving as a generic backend service.

Base URL

http://localhost:8080/api

Endpoints

GET /users

Retrieves a list of all registered users in the platform.

POST /users

Registers a new user. Required body: { "name": "...", "email": "..." }

GET /tasks/user/{userId}

Retrieves all tasks or goals assigned to a specific user.

POST /tasks/user/{userId}

Creates a new task for a user. Required body: { "title": "...", "description": "..." }

PATCH /tasks/{taskId}/status?status=COMPLETED

Updates the status of a specific task (e.g., PENDING, IN_PROGRESS, COMPLETED).
